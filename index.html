Welcome to Extempore

The High Performance Lisp Environment

Features

  • Aimed at users who need high performance code execution without a garbage collector
  • Originally developed to create dynamic audio and visual performances
  • Leverages LLVM's JIT compiler to produce optimised binary code
  • Multi-Platform Support (Windows, OSX, Linux)
  • Combines Scheme and XTLang in the environment, both LISP languages

Getting Started

The Extempore environment typically consists of two moving parts; the extempore process itself, and a text editor which can send code to the process in order to be evaluated. We support a number of different text editors, and installing extempore on most platforms is very straight forward.

For more instructions on how to get started, consult the guide.

Extempore in Action

;; load our libraries
(sys:load "libs/core/instruments.xtm")
(sys:load "libs/core/pc_ivl.xtm")

;; load our instrument
(bind-instrument fmsynth 
                 fmsynth_note_c fmsynth_fx)

;; we are going to play the 
;; instrument through the speakers
(bind-func dsp:DSP
  (lambda (in time chan dat)
    (fmsynth in time chan dat)))
(dsp:set! dsp)

;; play a random note on our instrument
(play-note (now) 
           fmsynth 
           (random 60 90) 
           80 44100)
